# API Híbrida Node.js

Proyecto académico desarrollado en equipo como parte de una plataforma tipo red social orientada a la comunidad estudiantil del Instituto

## Descripción
La plataforma permite a los estudiantes publicar productos y servicios para venta dentro de la institución, facilitando la conexión entre compradores y vendedores y ampliando el alcance de sus publicaciones dentro de la comunidad estudiantil.

La API REST fue desarrollada en Node.js y proporciona los servicios necesarios para la gestión de usuarios, publicaciones, mensajes y reportes dentro de la plataforma.

## Tecnologías utilizadas

- Node.js
- Express
- MySQL
- MongoDB
- JavaScript

## Mi participación

Desarrollo del módulo de reportes para publicaciones.

## Responsabilidades:

- Diseño e implementación de endpoints para la gestión de reportes.
- Desarrollo de la lógica de negocio asociada a los reportes de publicaciones.
- Integración con la base de datos.
- Pruebas y validación de las operaciones de consulta y administración de reportes.

## Funcionalidades implementadas

- Crear reportes de publicaciones.
- Consultar reportes registrados.
- Obtener información de un reporte específico.
- Actualizar el estado de un reporte.
- Eliminar reportes.

## Problema resuelto

Permitir a los usuarios reportar publicaciones que incumplieran las normas de la plataforma o contuvieran información inapropiada, proporcionando herramientas para su administración y seguimiento dentro del sistema.
